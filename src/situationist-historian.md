# シチュアシオニスト歴史学

シチュアシオニストの技法を1960年代のパリだけに留めず、歴史上の都市にもその射程を拡大できないだろうか。名付けて「シチュアシオニスト歴史学」だ。瞬間（モーメント）の組織化である「状況」やミクロな地政学と見ることも出来る「心理地理学」や宮中儀礼などが当てはまるであろう「スペクタクル」といった思考の枠組みを使って歴史的な都市空間と権力を考察する。

シチュアシオニスト歴史学は、あらゆる帝国的中心にある都市（長安、洛陽、ローマ、ペルセポリス、アレクサンドリア、コンスタンティノポリス、ハットゥシャ、イスファハーン、バグダート）を研究対象とすることが出来る。資本主義社会より王朝のほうが祭礼や謁見など＝スペクタクルが多かったように思える。王朝都市こそ、スペクタクル社会の原型的装置であり、資本主義とはその脱儀礼化・商品化にすぎない。

# 国家廃絶はアナキストの専売特許では無い

第一インターの対立を持ってしてマルクスを国家主義者と断罪するのはバクーニンの修辞法にすぎない。マルクスは明らかに国家権力をブルジョワの利益を守る暴力装置として嫌っていたし、自由な生産者のアソシエーションの共産主義社会像に関しては、バブーフやラッサールよりも遥かにアナキスト的である。ゆえに、19世紀社会主義の多くは国家の廃絶もしくは死滅の目標を共有していたのは確かである。

初期のコミンテルンの綱領にもしっかり「国家の完全な廃止への過渡期としての全世界ソヴィエト共和国の建設を任務とする...」などと書かれているのである（これは「映像の世紀」で知った話）。レーニンは思想上は国家死滅のヴィジョンを持っていたが、スターリン以後は国家の死滅は遠い未来に延長され、ついにブレジネフの全人民国家論によってマルクス主義と縁もゆかりもないリベラル国家論に成り下がってしまったのである。

なぜソヴィエトは国家を揚棄できなかったのか？柄谷行人曰く、国家の廃棄は交換様式と結びついているので非常に困難で、彼らは過度に楽観視していたから失敗したと指摘する。私はこの指摘を踏まえ、慣習＝生の様式としての国家に注目する。一般の国民は、医療制度から教育まで、生活を国家に依存してしまっている。

慣習として国家を前提としていると、国家が無くなっても国家の再生産を容易してしまう。だからこそ、生の様式を国家や権威に依存しない自律的なものに革命する必要がある。我々がオルタナティブを先に構築し、国家の存在意義そのものを無効化させることが、国家の廃絶とアナキズムの実現の道と考える。

# 再考 古代民主政 - 暴力的自由

古代民主政の方が体制としては強靭である。古代ギリシア、古代ローマ、古ゲルマン社会など、古代民主制においては、市民と戦士は同一であり、自らの民主政と自由を自らの暴力によって担保していた、自己完結的な体制であった。デヴィッド・グレーバーは『アナキスト人類学のための断章』で古代民主政のポリスの事を国家では無いので「都市国家」と呼ぶのは相応しくないと言っていたように、ポリスは部族的な戦士共同体なのである。

一方の近代民主主義（ブルジョワ民主主義）は、暴力が警察など抑圧的国家装置に集中しており、アメリカのように武装権が市民の手にあっても軍事組織と直結しておらず、イデオロギー装置によって支えられている近代民主主義はヘゲモニーしか担保がないという非常に脆弱な体制である、というか体制というべきではなく統治手法の一つでしかない。だから権威主義化は起きて当然の結果である。

フランス革命においては主権が王から国民へ臣民が国民に変わっただけであり、革命戦争では連盟兵ではなく徴兵が行われた。ロシア革命では党軍である労農赤軍が軍事を握っていたのでボルシェヴィキ独裁を産んだ。労兵ソヴィエト自体が軍事を自治的に統括していたらロシア革命の結果の条件もまた変わっていたであろう、これを実践したのがマフノ主義のウクライナ革命反乱軍であるが。

だから私はアナキスト革命においても、日常生活を全般に渡っての自治を行い自己決定する評議会権力は、各人が自発的に武装し共同で防衛にあたり、媒体となる官僚機構を再生産しないために、軍事・政治・生産を統合するべきと考えている。自己決定を行うためには暴力を他者に委ねてはいけないのだ。他者の支配を拒む事が出来るのは、支配を拒む為の力を持つ者のみである。従って、自由とは常に暴力的であり、その暴力を放棄した自由は、もはや自由ではない。ここで言う暴力とは、ソレルが『暴力論』で述べた様に、強制力（フォルス）に抵抗する暴力（ヴィオランス）の定義と一致する。

近代の自由は、しばしば暴力や侵害の否定として語られる。いわゆる消極的自由だが、それはリヴァイアサンたる国家に暴力を委ねて許された自由に過ぎない。また、社会自由主義における積極的自由などという概念は、自由の定義を能力や社会的条件へと変質させ、さらに国家依存にしてしまった。我々が賛美する、**自らの剣によって支配を拒むことで守られる自由**の事を、挑発を込めて「暴力的自由」と呼んでみようか。きっと近代の自由概念に対するアナキスト人類学的転倒を提起できるだろう。暴力的自由とは、抽象的な理念では無く、強い実体を伴う力である。
