# 前衛技術宣言 - Manifesto for Avant-Garde Tech

前衛芸術があるなら、前衛技術があっても良いではないか？

ダダ、シュルレアリスム、未来派、レトリスム、シチュアシオニスト。\
これらの前衛芸術運動が芸術の転倒による総合的変革を目指したように、\
技術もまた、存在様式の総体として自らを変革すべき時に来ている。

だが、技術が飛躍的に発達した近代になっても長い事、\
技術は問題解決の手段＝道具的理性としてのみ扱われてきた。\
技術が存在論的な意味を持つといふことは封ぜられ、\
人々はその重大な事実から目を背け続けている。

我々は断言する。技術は存在様式そのものである！と。\
ハイデガーは『技術への問い』でまさにその事実を突きつけた。

アプリ、クラウド、基幹系、OS、組み込み、コンパイラ。\
それらの技術はもはや世界の「開け」そのものを体現している。\
技術とは最早「使う」ものではない。生を規定する社会構造である。

前衛技術運動は、革命運動である。\
しかし、現代のサイファーパンクを名乗る革命家たちは、\
旧態依然として、技術を手段としてしか見ていない。

Web 3.0、ブロックチェーン、ハクティヴィズム、ビットコイン。\
これらのいわゆる暗号アナキズムと呼ばれるハッカー達の運動は、\
技術に「よって」アナキズム的解放を実現すると抜かしている。

我々は、技術よる解放ではなく、技術そのものの解放を目指す。\
問題解決ではなく、純粋な技術的探究心の遊戯性に依拠する。\
技術そのものをアナーキーにするのだ。

今日のインターネットの惨状の有様を見よ。\
検閲とアカウント凍結、通報という数タップの密告の応酬。\
テクノ全体主義はGAFAら大企業の手によって成立している。\
中国の情報統制はデジタルレーニニズムと呼ばれている。

ユーザーに不自由を強いる現代技術の低俗なあり方は、\
ハイデガーがゲシュテルと呼んで批判した時よりも、\
ドゥボールがスペクタクルと呼んで批判した時よりも、\
さらに深刻度を増してきている。最早革命に猶予はない。

テクネーとしての技術の本来性を取り戻そうではないか？\
技術の詩的な創造性、すなわちポイエーシスに回帰するのだ。

ドゥボールの「芸術家の否定が芸術の実現である」を転倒し、\
我々は「エンジニアの否定こそが技術の実現である」といふ。

エンジニアといふ資本の利益追求の奴隷にされた哀れな職業は、\
技術を生産性やマネジメントの枠に囲い込み、ゲシュテル化する。\
職能形態では技術の解放は満たされないどころか寧ろ反動となる。

既にエンジニアの諸君は、エンジニア性そのものを自己否定せよ！\
全ての技術者は、遊戯する自由なオタクとして、技術をするべきだ。\
金のために技術を手段として扱った段階で、技術は堕落してしまう。

このように抑圧が強化されゆく技術的・社会的情勢の中で我々は、\
技術を存在論的領域、アナキズム的次元まで高める必要がある。

実践として我々は、シチュアシオニストの戦術を踏襲する。\
資本主義的技術スペクタクルの転用（デトゥルヌマン）こそ、\
我々が前衛技術運動における主要な遊撃戦となる。

戦術は具体的には、無許可の逆コンパイルと流出である。\
著作権などアナキストには知ったことではないのだから。

また、ブラクラやジョークウイルスの開発も賛美する。\
利益を分取ろうとする資本主義的ランサムウェアとは異なり、\
他者の画面に介入する「状況の構築」として機能する。\
何より、純粋な技術的好奇心の表現形態であるから。

正直なコンパイラ、ターミナル操作、テキスト形式。\
ユーザーに自由を齎す技術作品の開発も進めるべきだ。\
これは各人の創造性に依拠する。

同時に革命運動としてのバイブスも高揚させる必要がある。\
閉塞した技術のゲシュテルを切り裂き、突破口を開くのだ。

前衛技術革命の暁には、凡ゆる技術は自由なる生と一体化し、\
エンジニア労働は消滅し、無政府共産主義が実現されるであろう。\
自由と発明に満ち溢れた技術は、遊戯と何ら変わらなくなる。

技術に存在論的解放を！\
アナーキー万歳！

# 無政府反動主義

科学技術（特に人工知能）の進歩は資本主義を加速させ人間の疎外と堕落を促進する。其の反動として愈々フォイエルバハの人間学を以ってアナキズムと反グローバリスム運動の大義と掲げられて然るべし時で在る。私は反動主義者だ、反動アナキストだ。諸君は此れを矛盾かと思ふかもしれないが、さうではない。即ちそれが意味するのは、文明以前の未分化な原始共同体の自由な生に意地でも留らうとせむ反動だ。資本主義の加速は必然的に矛盾を來すであらう故に、自由と自発的連帯への人間学的意志力に以って原遊動性を再誕させ回帰する革命の契機と為り得るのは、無政府反動主義しか有り得ない。

私は、文明の科学技術を放棄せよう、狩猟採集生活に戻らうと云ふ訳では無い。技術を転倒させたひのだ。今日の技術は、人を支配せやうとする技術である。検閲、フィルタンリング、検索アルゴリズム、業務管理システム。技術は利便性の為と云ひながら、其れは所詮は資本の自己増殖にとつての利便性に過ぎない。人間性の対する侮辱で有り、根本的な疎外形態が生じてゐる。此れを真に人間的な自由の為の形態へと転倒させやうと云ふのが、前衛技術運動である。

前衛技術運動とは、ハイデガーの『技術への問ひ』を思想的参照点と為し、シチュアシオニストの革命実践を継承しやうとせむ試みで在る。技術を単に道具的理性と扱つてゐる現状を批判し、「エンジニアの否定こそ技術の実現で在る」と云ふ命題を掲げ、技術の存在論的転倒を目指してゐる。現代の資本主義における技術の本質はゲシュテル、即ち存在を資源として配置し支配する構造に有る。前衛技術運動はこの“囲い込み”を破り、技術を再び世界のポイエーシスな“開示”の場として回復せむとする。
