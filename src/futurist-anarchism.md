# 未来派アナキスト宣言

1. 未来派アナキストは、偉大なる我々に無礼にも超越し君臨しようとする、クソ生意気な**権威と支配**の全ての抑圧を、唸る鋼鉄の**総破壊**で徹底的に叩き壊し、積極的に**超人**としての自己を創造する**自由な生**を希求する、理屈では説明の付きようがない生命の自己保存の本能的衝動、即ち**力への意志**の電撃的爆発である、神聖にして正しいアナキスト世界革命戦争——腐り切ったこの世界に衛生を齎す唯一の処方——を恐れずに公然と賛美しよう。

2. アナキスト世界革命戦争の完徹——地球上から190以上もの凡ゆる国家主権とその資本主義的搾取機構を、黒光りする機関銃で爆破粉砕し塵一つ残さず蒸発させる——には、国家の強制力を凌駕する**強大な暴力**を要する。小難しい退屈なインテリの理論や考察では無く、破壊への情熱＝創造への情熱の反逆精神を共有し、我々を闘争に駆り立てる原動力である**ゼネストの神話**と呼ばれる普遍的な大きな物語を称揚し、自由意志と自発性によって力強く結束する。

3. 絶え間ない総破壊の実践と同時に、権力の真空状態における官僚機構の再生産を防ぐ為に、虚栄と嘘に飾られ**スペクタクル化**されたデジタル管理社会＝資本主義体制に生活を依存する事によって生じる**疎外を克服**しなければならない。自由な生のエネルギーが十二分に発揮される革命的な**状況を構築**する為に、資本主義的なメディア広告を逆手に取って**転用**（デトゥルヌマン）して愉快な悪戯を仕掛け、我々の目的意識的な意志力と実践によって社会構造を根本から転倒させ、日常生活の在り方そのものを「今・ここ」の瞬間から永久に、瞬時に、革命する！

4. 古き支配の腐敗した慣習や女々しき感傷や博物館の伝統を断固として拒絶し、技術の冷徹な**光速的加速**によって日々増大する自動車と電車の金属的な騒音、新宿や渋谷の駅のターミナルを縦横無尽に横断する都市プロレタリアートの**有象無象の群集**を称揚する。もし彼らの怒りと欲望が一気に爆発したら、東京——天皇陛下wのおわします厳かな首都——は果たして一体全体どうなるのであろうか！創造しただけでも興奮する。彼らの怒りを扇動し、欲望を刺激するのが我々の革命的任務である。そして際限なき自由とアナーキーの中で民衆は自発的に**自己組織化**し、評議会は熱狂と祝祭に包まれ「下から上へ」と連邦的に増殖するであろう！
