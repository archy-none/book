# 未来派アナキスト宣言

1. 未来派アナキストは、全ての権威と支配の抑圧を暴力的に**総破壊**し、自由な生のエネルギーと**力への意志**の爆発である、神聖にして正しいアナキスト世界革命戦争——腐り切ったこの世界に衛生を齎す唯一の処方——を公然と力強く賛美しよう。

2. アナキスト世界革命戦争の完徹には、国家の強制力を凌駕する強大な暴力を要する。我々は自由意志と自発性、破壊への情熱＝創造への情熱に基づき、我々を闘争に駆り立てる原動力である**神話**——普遍的な大きな物語——を共有し、強固に団結する。

3. 絶え間ない総破壊の実践と同時に、権力の真空状態における官僚機構の再生産を防ぐ為に、生活様式を**スペクタクル化**された資本主義体制に依存し事による疎外を克服し、自由な生の**状況の構築**を進める必要がある。すなわち、日常生活の革命である。

4. アナキズム社会の具体的実装は、日常生活の全般——軍事から生産やケアに至るまで——を統合的に直接民主政の原則に基づいて自治する**評議会**によって体現される。評議会は、自由な諸個人によって「下から上へ」とボトムアップに形成され、流動性的に自己組織化されるアソシエーションである。
