# 未来派アナキスト宣言

1. 未来派アナキストは、全ての権威と支配の抑圧を暴力的に**総破壊**し、自由な生を欲する自己保存の野生的本能と**力への意志**との電撃的爆発である、神聖にして正しいアナキスト世界革命戦争——腐り切ったこの世界に衛生を齎す唯一の処方——を公然と力強く賛美しよう。

2. アナキスト世界革命戦争の完徹には、国家の強制力を凌駕する強大な暴力を要する。小難しいインテリの理論では無く、破壊への情熱＝創造への情熱の精神を共有し、我々を闘争に駆り立てる原動力である**神話**——普遍的な大きな物語——を共有し、自由意志と自発性によって結束する。

3. 絶え間ない総破壊の実践と同時に、権力の真空状態における官僚機構の再生産を防ぐ為に、生活様式を**スペクタクル化**された資本主義体制に依存する事に生じる**疎外を克服**しなければならない。自由な生のエネルギーが十二分に発揮される**状況を構築**する為に、我々の意志の力と集団的な実践によってこの社会を転倒させ、我々は日常生活の在り方そのものを「今・ここで」永久に革命する。

4. アナキズム社会——自分の事は自分で決めるという至極普遍的な理想——の具体的実装は、自由な諸個人によって「下から上へ」と連邦的にボトムアップに形成され、直接民主政と原則に基づいて自己組織化する**評議会**によって体現される。評議会は、日常生活の全般——生産や消費から教育やケアに至るまで——を統合的に自治する。
