# 未来派アナキスト宣言

1. 未来派アナキストは、全ての権威と支配の抑圧を暴力的に総破壊し、自由な生のエネルギーと力への意志の爆発である、神聖にして正しいアナキスト世界革命戦争——腐り切った世界へ衛生を齎す唯一のもの——を公然と力強く賛美しよう。

2. アナキスト世界革命戦争の完徹には、国家の強制力を凌駕する強大な暴力を要する。我々は自由意志と破壊への情熱＝創造への情熱に基づき、闘争の原動力たる革命神話を友愛的に共有し強固に団結する。

3. 絶え間ない総破壊の実践と同時に、権力の真空状態における官僚機構の再生産を防ぐ為に、生活様式を体制に依存する事の無い自由な生の状況の構築を進める必要がある。すなわち、日常生活の革命である。
