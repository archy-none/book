# アソシエーション

イギリスのいわゆる空想的社会主義者たちやプルードンはマルクス・エンゲルスによって批判されたが、この場合マルクスの批判には妥当性があった。なぜなら彼らは交換様式AやCによってアソシエーションを作ろうとしていたからだ。そうではなくて、もちろんラッサールやレーニンの国家社会主義のように交換様式Bに依拠するのでもなくて、共産主義のアソシエーションはバクーニンが唱えた偶像や既存社会の総破壊によって、偶像崇拝の否定によって、つまり現体制を無視して権藤成卿のように穏健なコミューン運動を展開するのではなくて、抑圧的なこの社会を積極的に破壊することによって交換様式Aの高次の回復、抑圧・搾取的な人類文明のアウフヘーベンである交換様式Dによって成し遂げなければならなかったからだ。

だからアソシエーションを作ろうとしていたマルクスのプルードン批判は妥当だったと言わざるを得ないが、マルクスのバクーニン批判は妥当性を有していなかったのだ。アナキズムは決して一部のマルクス主義者がうぬぼれて主張するように一筋縄で論破できるようなまとまった主張ではない、アナキズムという人間の根源的欲求にもとづく哲学は定式化と表面的な論破を絶えずすり抜けるし、そう容易な反論を行いえないものである。

マルクスはプロレタリア独裁によって資本主義体制を破壊し、アソシエーションを作ることを主張した。この主張はレーニンが実践したこととは似ても似つかないものである。第一インターナショナルでバクーニンと激論をたたかわせたマルクスは、もしレーニンを前にしたらバクーニンに対して以上にきっと辛辣な批判を浴びせずにはおかないだろう。

しかし、マルクスはプロレタリア独裁を議会制ではなくて直接民主制の自由な共同体として構想したはいうものの、権力分散の必要性を認めずにむしろ中央集権のコミューン国家を主張した。マルクスは中央集権の直接民主制を実現できると考えていたのだ。そして、闘争の手段としてブルジョア議会への参加を肯定した。

これがのちに第二インターのカウツキー主義（議会制国家社会主義）を生み出し、事実上のラッサール回帰を発生させてレーニン主義を準備したことは明らかではないだろうか。しかしそれは、そもそもマルクスのいう中央集権国家が自由な直接民主制によって運営されるということが、人口・面積の大きい世界革命根拠地ではそもそも不可能だということから帰結する。一定以上の領域と人口を抱える革命政権を、分権・連邦制ではなくて中央集権によって管理するというのであれば、直接民主制ではなくて議会制民主主義になってしまうことはカウツキーのように明白ではないか！

そして、カウツキーをあれだけ批判したレーニンもカウツキー以上に抑圧の酷い国家主義体制を生み出したのだ。ソヴィエトという名前を冠しながら。

バクーニンの優位性は証明されたのではないだろうか。マルクスも共産主義のアソシエーションを真剣に作ろうとしたが、マルクスの方法でアソシエーションを作ることは不可能だったのだ。バクーニンが指摘したように、人類が資本の支配から脱却する集産主義は、分権的共同体でなければ成立できない。

# 普遍宗教と交換様式D

古代エジプト王朝という交換様式Bの圧政を否定して出来たのがユダヤ教で、ユダヤ教の硬直・権威主義化を批判し、かつ「金持ちが神の国に入るよりも、らくだが針の穴を通る方がまだ易しい」（ルカ18章25節)とイエスが言ったように高利貸しなどの貨幣経済という交換様式Cやローマ帝国の奴隷に対する交換様式Bを否定することで広がっていったのがキリスト教だ。

これらの宗教は偶像崇拝を禁止したことによって今まで人々を束縛していた神を打倒した。普遍宗教とは宗教を否定する宗教だった。超人思想を展開したニーチェのように神を否定した先に新しい神を再肯定することによって、疎外されて死んでいった精神（世界宗教）は堕落から復活（ルネサンス）し、交換様式BやCを破壊しつつも、支配関係が存在しなかった原始の過去にそのまま回帰するのではなくて否定しながらに契機として保存しながら包摂（揚棄 Aufheben）することで生命ある宗教が生まれてくるのだ。

それは文明が存在せず、したがってまた抑圧も存在しなかった交換様式Aの原始状態がそのまま回帰してくることではなくて、最初には疎外された形態で文明が誕生したが、そこにある国家や階級社会という制約を打ち破って文明というものを否定的に包摂することで、個人の生を普遍的に肯定する高次の形態で文明がきっと起源へと回帰していくはずだ。なぜなら疎外は矛盾を生み出し、矛盾は科学的理論として把握されるだけではなくて、魂の叫びとして、生きている奴隷たちの反発をついには革命へと押し上げずにはおかないからだ。抑圧的な社会が本当の意味で安定することは科学的に考えなくても実存感性としてあり得ないのだ。

# 再考 古代民主政

古代民主政の方が体制としては強靭である。古代ギリシア、古代ローマ、古ゲルマン社会など、古代民主制においては、市民と戦士は同一であり、自らの民主政と自由を自らの暴力によって担保していた、自己完結的な体制であった。デヴィッド・グレーバーは『アナキスト人類学のための断章』で古代民主政のポリスの事を国家では無いので「都市国家」と呼ぶのは相応しくないと言っていたように、ポリスは部族的な戦士共同体なのである。

一方の近代民主主義（ブルジョワ民主主義）は、暴力が警察など抑圧的国家装置に集中しており、アメリカのように武装権が市民の手にあっても軍事組織と直結しておらず、イデオロギー装置によって支えられている近代民主主義はヘゲモニーしか担保がないという非常に脆弱な体制である、というか体制というべきではなく統治手法の一つでしかない。だから権威主義化は起きて当然の結果である。

フランス革命においては主権が王から国民へ臣民が国民に変わっただけであり、革命戦争では連盟兵ではなく徴兵が行われた。ロシア革命では党軍である労農赤軍が軍事を握っていたのでボルシェヴィキ独裁を産んだ。労兵ソヴィエト自体が軍事を自治的に統括していたらロシア革命の結果の条件もまた変わっていたであろう、これを実践したのがマフノ主義のウクライナ革命反乱軍であるが。

だから私はアナキスト革命においても、日常生活全般を自治し自己決定する評議会権力は、やはり自律的に暴力を管理し、軍事と政治と生産が一体化するべきであると考えている。自己決定を行うためには暴力を他者に委ねてはいけないからね。他者の支配を拒むことができるのは、支配を拒むための力を持つ者のみである。したがって、自由とはつねに暴力的であり、その暴力を放棄した自由は、もはや自由ではない。

近代の自由は、しばしば暴力や侵害の否定として語られる。いわゆる消極的自由だが、それはリヴァイアサンたる国家に暴力を委ねて許された自由に過ぎない。また、社会自由主義における積極的自由などという概念は、自由の定義を能力や社会的条件へと変質させ、さらに国家依存にしてしまった。我々が賛美する、自らの剣によって支配を拒んで守られる自由の事を、挑発を込めて「暴力的自由」と呼んでみようか。
